# Counter

Минимальное мобильное приложение-счётчик на React Native и Expo.

## Возможности

- Крупный текст с текущим значением по центру экрана (начальное значение — 0)
- Кнопка **+** — увеличивает значение на 1
- Кнопка **Сброс** — сбрасывает значение в 0

## Запуск

```bash
npm install
npm start
```

После запуска откройте приложение в Expo Go на телефоне (отсканируйте QR-код) или в эмуляторе (Android/iOS).
